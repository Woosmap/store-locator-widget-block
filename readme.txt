Block for Woosmap Store Locator Widget
====================

This document provides an overview of the "Block for Woosmap Store Locator Widget" plugin for WordPress, detailing its features, installation, configuration, and frequently asked questions.

## Contributors
- camilo
- galela

## Features
- Integrates Woosmap Store Locator Widget JS into WordPress block editor (Gutenberg).
- Requires a Woosmap account for configuration of API Key.
- Supports full configuration for Store Locator Widget.
- Includes end-to-end testing setup with Playwright.

## Description
The plugin adds a Woosmap Store Locator Widget block to the WordPress block editor, leveraging Woosmap Map for map display. It requires configuration with a Woosmap account and a Public API Key.

## Installation
1. Install and activate the plugin via the WordPress plugin installer.
2. Configure access to the Woosmap Map JS API following the provided instructions.
3. Start using the Woosmap Store Locator Widget block in your posts and pages.

## Configuration
Configuration involves creating a Woosmap account and generating the necessary Public API Key.
